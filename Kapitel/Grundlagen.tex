The Visual Studio 2013 project templates use Bootstrap, a layout and theming
framework created by Twitter. Bootstrap uses CSS3 to provide responsive design,
which means layouts can dynamically adapt to different browser window sizes. 
You can also use Bootstrap's theming feature to easily effect a change in the
application's look and feel. By default, the ASP.NET Web Application template in
Visual Studio 2013 includes Bootstrap as a NuGet package.
