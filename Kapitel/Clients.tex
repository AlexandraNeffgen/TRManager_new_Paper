
\subsection{Anforderungen}
Die Daten werden vom Backend über einen Restful-Client in JSON empfangen und dem
Benutzer in einer grafischen Oberfläsche entsprechend prädentiert. Dabei werden
die Daten als Objekte in der Anwendung gehalten. Um ausfallsicher zu arbeiten
wird jede Änderung der Daten direkt an das Backend gesendet.
